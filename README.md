# Cake Configure Completion

Plugin provides the following features:

Plugin provides the following features:
- Auto completion for Configure::read and Configure::write
- Jump to definition for configuration strings

Key                  | Value
-------------------- | --------------------
Plugin Url           | https://plugins.jetbrains.com/plugin/15620-cake-configure-completion
ID                   | CakeConfigureCompletion

## Install
- Install the plugin by going to `Settings -> Plugins -> Marketplace` and then search for `Cake Configure Completion`

## Usage

When typing configuration paths for `Configure::read("")` or `Configure::write("")` it will automatically suggest configuration keys for autocompletion.

By Ctrl+Click the configuration key, it will jump to the configuration declaration.
